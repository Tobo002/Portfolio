# Thomas Boyle — Portfolio

This repository contains my professional portfolio website. The site presents an introduction to who I am, my working approach, and previews of my projects.

## Overview

The site is structured as a static website with a primary landing page and dedicated pages for background and working practice. Key sections include:

- Home — a concise introduction and links to project previews
- About — a short biography and interests
- How I Work — my approach, tools, and experience
- Case Studies — project preview cards with placeholder pages

I keep project assets, page templates, and planning notes in this repository. The published site is stored in the `_site/` directory and includes the page files and images used by the site.

## Project Structure (summary)

- `_site/` — published site files (HTML, CSS, JS, images)
   - `index.html` landing page
   - `photos/` contains images used in the site
   - `pages/` includes other pages
   - `css/` and `js/` contain the style and script files

## Technologies

The site is implemented with HTML, CSS, and JavaScript and uses Bootstrap for responsive layout and components. I deploy the site with Netlify and keep source control in GitHub.

## Contact

Contact links are available in the site footer (LinkedIn and GitHub).

## Status

This site is a work in progress: content will be refined and case studies expanded over time.
